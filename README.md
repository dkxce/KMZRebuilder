# KMZ Rebuilder

KMZRebuilder allow to create/edit/manage/split/merge/import/export/explore yours POI

This is a special tool to edit kmz files.
Add, remove, rename placemarks & points.
Change icons.
Move points.
Split and merge kmz files.
