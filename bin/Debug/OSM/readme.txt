1. Grab zip from https://github.com/ErshKUS/osmCatalog
2. Save `catalog.json` from zip here
3. Save `dictionary_ru.json` as `dictionary.json` here
4. Unzip poi_maker png files to any folder and zip them into `poi_marker.zip`
5. Save `poi_marker.zip` to ..\mapicons\poi_marker.zip

Now you can use import from OSM XML
